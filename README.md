## How to Install

1. Clone this repository:

```bash
git clone https://github.com/blahkheart/ethrwanda.git
```

2. Go into project directory:

```bash
cd ethrwanda
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
